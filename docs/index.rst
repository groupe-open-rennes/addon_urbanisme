À propos
======================

Cette extension de `geOrchestra <https://www.georchestra.org/fr/projet.html/>`_ permet de consulter une note de renseignement d'urbanisme ou une fiche d'informations "zonage du PLU". L'information peut également être exportée sous forme de document PDF.
